My experience with the creative coding challenge

The live coding challenge was the creation and modification of the classic snake game. The object-oriented programming used in this project helped me as a student by helping me create a formula for different things in life. Instead of fully improvising, I instead only improvise if necessary. For me, the most important aspect of this project was my implementation of object-oriented programming. The actual application of this method of the program I found was a valuable learning experience. An example of this application is in my version of the snake game where I used object-oriented programming to create obstacles.
