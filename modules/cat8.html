Microbit

My experience with microbit

The micro bit was a fun project that introduced me to block code. In the future, due to its relatively different style, I can apply the fundamental concepts of block code to future projects to tackle the problem from a different perspective. The most important part of this project was the new experience with using block code, which is an important skill that I will need to continue to apply to gain more experience. As mentioned before, version control was also very important in this project to ensure that we did not accidentally break the entire code.
